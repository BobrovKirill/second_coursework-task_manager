# API Documentation

## Users & Auth

**Дата:** 19.05.2026

---

### Пользователи (`/users`)

| Эндпоинт           | Метод  | CRUD | Автор   | Описание                                     | Примечания                  |
|--------------------|--------|------|---------|----------------------------------------------|-----------------------------|
| `/users/`          | POST   | Да   | Кирилл  | Создание нового пользователя                 | -                           |
| `/users/{user_id}` | GET    | Да   | Кирилл  | Получение пользователя по ID                 | -                           |
| `/users/`          | GET    | Да   | Кирилл  | Получение списка пользователей               | `skip`, `limit` (пагинация) |
| `/users/{user_id}` | DELETE | Да   | Кирилл  | Удаление пользователя                        | -                           |
| `/users/me`        | GET    | -    | Кирилл  | Получение профиля текущего пользователя      | Поддерживает `?project_id=` |
| `/users/me`        | PATCH  | -    | Кирилл  | Обновление профиля текущего пользователя     | Поддерживает `?project_id=` |
| `/users/avatar`    | POST   | -    | Кирилл  | загрузка аватара в хранилище, возвращает url | -                           |

**Особенности эндпоинтов `/users/me`:**

- При передаче query-параметра `project_id` дополнительно возвращается:
  - Поле `role` — роль пользователя в проекте
  - Поле `permissions` — права пользователя в данном проекте

### Авторизация (`/auth`)

| Эндпоинт       | Метод | CRUD | Автор   | Описание                        | Примечания |
|----------------|-------|------|---------|---------------------------------|----------|
| `/auth/login`  | POST  | -    | Кирилл  | Авторизация пользователя        | Возвращает JWT access token |

**Тело запроса (`LoginRequest`):**
- `email`
- `password`

**Возможные ошибки:**
- `401 Unauthorized` — Неверный email или пароль
- `400 Bad Request` — Пользователь неактивен (`is_active = false`)

### Задачи (`/projects/{project_id}/tasks`, `/tasks`)

| Эндпоинт                        | Метод  | CRUD | Автор  | Описание                        | Примечания                                                      |
|---------------------------------|--------|------|--------|---------------------------------|-----------------------------------------------------------------|
| `/projects/{project_id}/tasks`  | POST   | Да   | Ксения | Создание задачи в проекте       | Сейчас доступно владельцу проекта                               |
| `/projects/{project_id}/tasks`  | GET    | Да   | Ксения | Получение списка задач проекта  | Доступно владельцу и участникам проекта                         |
| `/tasks/{task_id}`              | GET    | Да   | Ксения | Получение одной задачи          | Доступно владельцу и участникам проекта                         | 
| `/tasks/{task_id}`              | PUT    | Да   | Ксения | Редактирование задачи           | Доступно владельцу проекта, admin, organizer и создателю задачи |
| `/tasks/{task_id}`              | DELETE | Да   | Ксения | Удаление задачи                 | Доступно владельцу проекта, admin, organizer и создателю задачи |

**Возможные ошибки:**
- `403 Forbidden` — нет доступа к проекту или задаче
- `404 Not Found` — проект или задача не найдены
